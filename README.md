# Morse-Signal
This tool convert an input alphanumeric character into Morse code

## How to
$ brew install portaudio sdl_mixer<br>
$ clang++ morse.cpp $(sdl-config --cflags --libs) -lSDL -lSDL_mixer<br> 
$ ./a.out<br>
&gt;&gt; [input your favorite words :)]
