# Morse-Signal
This tool convert an input alphanumeric character into Morse code

## How to
$ brew install portaudio sdl_mixer
$ clang++ morse.cpp $(sdl-config --cflags --libs) -lSDL -lSDL_mixer 
$ ./a.out
>> [input your favorite words :)]
